# Installing ELG on macOS

1. Download and open `ELG.dmg`.
2. Drag **ELG** into the **Applications** folder.
> You can now delete the file you downloaded.
3. Try to open the app.
4. After being denied access, click **Done**.
5. Go to **System Settings > Privacy & Security**.
6. Scroll all the way down. Under **Security** you will find
   `"ELG" was blocked to protect your computer.` Next to it, select **Open Anyway**.
7. Confirm with **Open Anyway**.
8. Select **Use Password...** and enter your password.

> ELG is not yet code signed / notarized, which is why macOS blocks it on first launch.
> These steps are only needed the first time you open the app.
